# JavaScript Security Challenges

A level-based JavaScript project focused on DOM manipulation, browser storage, and HTTP requests.

## Levels Overview
- Level 01–07: DOM inspection & basic JavaScript
- Level 08–10: localStorage & cookies
- Level 11–13: HTTP requests, API responses, token extraction

## How It Works
1. Open index.html
2. Select a level
3. Read the task description
4. Solve using JavaScript and browser DevTools

## Technologies
- HTML
- CSS
- JavaScript

## Purpose
This project is built to practice real-world client-side security and JavaScript behavior.
